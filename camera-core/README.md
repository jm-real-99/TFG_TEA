# TFG_TEA
### Juan Manuel Real Domínguez

### OBJETIVO
Este servicio tiene como objetivo analizar las expresiones faciales del paciente con el fin de complementar los datos a la aplicación, además de poder evaluar si existe una mejora cognitiva y expresiva mediante la terapia de imitación.
### ORGANIZACIÓN 
Como se puede apreciar en la estructura de este módulo, tenemos distintas carpetas, aquí podrás encontrar qué significa cada cosa:

- **ejemplos:** Con la finalidad de ir haciendo un seguimiento en caso de querer retroceder, aquí podremos encontrar los scripts que he usado como referencia o las distintas etapas en la que he ido construyendo el script principal.
- **modelos:** Aquí encontraremos los modelos ya entrenados o los recursos para necesario para hacerlo. 
- **reconocimiento_emociones.py:** Como se encuentra en el directorio raíz, es el fichero principal.

Para información más detallada en cada fichero hay una breve explicación del script.
